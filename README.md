# Google Keep Clone

## Overview
**Google Keep Clone** is a lightweight, web-based note-taking application built with **HTML, CSS, and JavaScript**. It replicates the essential functionality of Google Keep, allowing users to quickly create, view, and manage notes in a clean, card-based interface.  

The application provides a **fast and accessible way to capture ideas and reminders** directly in the browser, without the need for heavy software or logins.

## Problem
Many users need a **simple and immediate method to capture short-term notes, ideas, or reminders**. Existing solutions often require accounts, heavy apps, or complex interfaces, which slow down the process of recording information quickly.  

This project addresses that problem by offering a **minimalistic and intuitive note-taking system** that runs entirely in the browser.

## Solution
**Built** a responsive note-taking application **by implementing dynamic DOM manipulation and JavaScript-driven interactions**, **enabling users to create, display, and remove notes instantly in a clean, organized layout.**

## Features
- Create notes instantly from the browser  
- Display notes in a card-style layout similar to Google Keep  
- Delete notes to keep the workspace organized  
- Responsive design suitable for all screen sizes  
- Lightweight, no external libraries or frameworks required  

## Technologies Used
- **HTML5** – for structure and markup  
- **CSS3** – for styling and layout  
- **JavaScript (Vanilla JS)** – for functionality and interactivity
